<p><b>Bitbucket</b> is a <a href="/wiki/Git_(software)" class="mw-redirect" title="Git (software)">Git</a>-based <a href="/wiki/Source_code" title="Source code">source code</a> <a href="/wiki/Repository_(version_control)" title="Repository (version control)">repository</a> <a href="/wiki/Shared_web_hosting_service" title="Shared web hosting service">hosting service</a> owned by <a href="/wiki/Atlassian" title="Atlassian">Atlassian</a>.  Bitbucket offers both commercial plans and free accounts with an unlimited number of private repositories.
</p>

<div class="mw-heading mw-heading2"><h2 id="Services">Services</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bitbucket&amp;action=edit&amp;section=1" title="Edit section: Services"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></div>
<div class="mw-heading mw-heading3"><h3 id="Bitbucket_Cloud">Bitbucket Cloud</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bitbucket&amp;action=edit&amp;section=2" title="Edit section: Bitbucket Cloud"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Bitbucket Cloud (previously known as Bitbucket) is written in <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a> using the <a href="/wiki/Django_(web_framework)" title="Django (web framework)">Django</a> <a href="/wiki/Web_framework" title="Web framework">web framework</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
</p><p>Bitbucket is mostly used for code and <a href="/wiki/Code_review" title="Code review">code review</a>. Bitbucket supports the following features:
</p>
<ul><li>Pull requests with code review and comments<sup id="cite_ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Bitbucket Pipelines,<sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup> a continuous delivery service</li>
<li>Two-step verification and required two-step verification<sup id="cite_ref-4" class="reference"><a href="#cite_note-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup><sup id="cite_ref-infoq.com_5-0" class="reference"><a href="#cite_note-infoq.com-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup></li>
<li>IP whitelisting<sup id="cite_ref-infoq.com_5-1" class="reference"><a href="#cite_note-infoq.com-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Merge Checks<sup id="cite_ref-6" class="reference"><a href="#cite_note-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Code search (Alpha)<sup id="cite_ref-7" class="reference"><a href="#cite_note-7"><span class="cite-bracket">&#91;</span>7<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Git Large File Storage (LFS)<sup id="cite_ref-infoq.com1_8-0" class="reference"><a href="#cite_note-infoq.com1-8"><span class="cite-bracket">&#91;</span>8<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Documentation, including automatically rendered README files in a variety of Markdown-like file formats</li>
<li>Issue tracking<sup id="cite_ref-9" class="reference"><a href="#cite_note-9"><span class="cite-bracket">&#91;</span>9<span class="cite-bracket">&#93;</span></a></sup></li>
<li><a href="/wiki/Wiki" title="Wiki">Wikis</a><sup id="cite_ref-10" class="reference"><a href="#cite_note-10"><span class="cite-bracket">&#91;</span>10<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Static sites hosted on Bitbucket Cloud: Static websites have the bitbucket.io domain in their URL<sup id="cite_ref-11" class="reference"><a href="#cite_note-11"><span class="cite-bracket">&#91;</span>11<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Add-ons and integrations<sup id="cite_ref-12" class="reference"><a href="#cite_note-12"><span class="cite-bracket">&#91;</span>12<span class="cite-bracket">&#93;</span></a></sup></li>
<li>REST APIs to build third-party applications which can use any development language<sup id="cite_ref-13" class="reference"><a href="#cite_note-13"><span class="cite-bracket">&#91;</span>13<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Snippets that allow developers to share code segments or files<sup id="cite_ref-atlassian.com_14-0" class="reference"><a href="#cite_note-atlassian.com-14"><span class="cite-bracket">&#91;</span>14<span class="cite-bracket">&#93;</span></a></sup></li>
<li>Smart Mirroring<sup id="cite_ref-15" class="reference"><a href="#cite_note-15"><span class="cite-bracket">&#91;</span>15<span class="cite-bracket">&#93;</span></a></sup></li></ul>

<p><b>Bitbucket Server</b> (formerly known as <b>Stash</b><sup id="cite_ref-18" class="reference"><a href="#cite_note-18"><span class="cite-bracket">&#91;</span>18<span class="cite-bracket">&#93;</span></a></sup>) is a combination <a href="/wiki/Git" title="Git">Git</a> server and web interface product written in <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a> and built with <a href="/wiki/Apache_Maven" title="Apache Maven">Apache Maven</a>.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19"><span class="cite-bracket">&#91;</span>19<span class="cite-bracket">&#93;</span></a></sup> It allows users to do basic Git operations (such as <a href="/wiki/Code_review" title="Code review">reviewing</a> or merging code, similar to <a href="/wiki/GitHub" title="GitHub">GitHub</a>) while controlling read and write access to the code. It also provides integration with other <a href="/wiki/Atlassian" title="Atlassian">Atlassian</a> tools.<sup id="cite_ref-20" class="reference"><a href="#cite_note-20"><span class="cite-bracket">&#91;</span>20<span class="cite-bracket">&#93;</span></a></sup>
</p><p>Bitbucket Server is a <a href="/wiki/Commercial_software" title="Commercial software">commercial software</a> product that can be <a href="/wiki/Licensed" class="mw-redirect" title="Licensed">licensed</a> for running on-premises.<sup id="cite_ref-license_21-0" class="reference"><a href="#cite_note-license-21"><span class="cite-bracket">&#91;</span>21<span class="cite-bracket">&#93;</span></a></sup> Atlassian provides Bitbucket Server for free to <a href="/wiki/Open-source_software" title="Open-source software">open source</a> projects meeting certain criteria, and to organizations that are non-profit, non-government, non-academic, non-commercial, non-political, and secular. For academic and commercial customers, the full source code is available under a developer source license.<sup id="cite_ref-license_21-1" class="reference"><a href="#cite_note-license-21"><span class="cite-bracket">&#91;</span>21<span class="cite-bracket">&#93;</span></a></sup>
</p>
<div class="mw-heading mw-heading2"><h2 id="History">History</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bitbucket&amp;action=edit&amp;section=4" title="Edit section: History"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Bitbucket was previously an independent <a href="/wiki/Startup_company" title="Startup company">startup company</a>, founded by Jesper Nøhr in 2008.<sup id="cite_ref-nabble.com_22-0" class="reference"><a href="#cite_note-nabble.com-22"><span class="cite-bracket">&#91;</span>22<span class="cite-bracket">&#93;</span></a></sup> On 29 September 2010, Bitbucket was acquired by <a href="/wiki/Atlassian" title="Atlassian">Atlassian</a>.<sup id="cite_ref-itwire_23-0" class="reference"><a href="#cite_note-itwire-23"><span class="cite-bracket">&#91;</span>23<span class="cite-bracket">&#93;</span></a></sup> In September 2015, Atlassian renamed their Stash product to Bitbucket Server.<sup id="cite_ref-24" class="reference"><a href="#cite_note-24"><span class="cite-bracket">&#91;</span>24<span class="cite-bracket">&#93;</span></a></sup> In July 2016, Bitbucket added support for Git Large File Storage (LFS).<sup id="cite_ref-infoq.com1_8-1" class="reference"><a href="#cite_note-infoq.com1-8"><span class="cite-bracket">&#91;</span>8<span class="cite-bracket">&#93;</span></a></sup>
In March 2020, Bitbucket Server controversially dropped support for viewing three way diffs<sup id="cite_ref-25" class="reference"><a href="#cite_note-25"><span class="cite-bracket">&#91;</span>25<span class="cite-bracket">&#93;</span></a></sup> and in July 2020, Bitbucket Cloud removed support for its original repository format <a href="/wiki/Mercurial" title="Mercurial">Mercurial</a>.<sup id="cite_ref-26" class="reference"><a href="#cite_note-26"><span class="cite-bracket">&#91;</span>26<span class="cite-bracket">&#93;</span></a></sup>
</p>
